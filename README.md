# Deep Sea Explorers

An interactive exploration game of the deep dark sea.

## Overview
This software is a story side scroller that tells a story about a team of navigators and geologists on their mission to the deep sea. Their mission, collect a million dollars worth of expensive minerals and samples of coral while avoiding the hazards that harm the
submarine. If they go through too many hazards, submarine explodes. 

Lead these brave navigators and geologists through the hazards of the deep dark seafloor and make sure they make it home alive.

## Datasets
- cells.csv
- hazards.csv
- poi.csv
- life.csv
- resources.csv

## Installation and running
1. clone into directory
2. cd client/
3. npm install
4. cd ../server
5. npm install
6. cd ../client
7. npm run dev
